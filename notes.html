Block and Inline Elements

Block Elements

Block elements make an invisible block around themselves. This box has a height, width, and margins above/below it.

Examples include:p, ol, ul, h1-6, video, div, table.

https://developer.mozilla.org/en-US/docs/Web/HTML/Block-level_elements#Elements

Inline Elements

Inline elements don't create a box around themselves.

Examples include:br, em, strong, mark, a, img, span.

https://developer.mozilla.org/en-US/docs/Web/HTML/Inline_elements#Elements

Files and Relative URLs

File doesn't have to be on a server. It can be on the same computer as the code.

Instead, write "img src="(ImageName.jpg)" alt="Name of Image"". This is called a Relative URL.

DOCTYPE Tag

Tells the browser what type of HTML the code is written in.

Looks like: "!DOCTYPE html".

It goes on top of every HTML document.

Head and Body Elements

The body "body" contains all of the stuff that would be put into a regular HTML code.

The head "head" contains pieces of important information about the code. The only required part is the title of the document, and it is put in as a "title" element. Style information is also put in here.

Both are in "html".
